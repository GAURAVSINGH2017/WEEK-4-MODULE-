This is an week 4 c language module . 

<b>The following topics are covered in this module:-</b>

1. wap in c to count the total number of digits present in a number.
2. wap in c to reverse the digits of the number and check whether it is pallindrome or not.
3. wap in c to calculate the sum of the digits of the number.
4. wap in c to calculate how many even and odd digits are present in a given number.
5. wap in c to print the largest and smallest digit present in the  given number.
6. wap in c to print how many prime numbers are present between  a two positive integer.
7. wap in c to print the sum of all proper divisor of a number.
8. wap in c to print all the factors of the number in ascending order.

<b> All these programs are included in that module by proper explanation and syntax </b>
